# 4S Li-Ion Battery Management System

This project was developed as part of the **EEE316 Microprocessors** course.  
The main objective of the project is to design and implement a basic **4-cell Li-Ion Battery Management System (BMS)** using a microprocessor-based control structure.

## Project Overview

The system focuses on monitoring a 4-cell Li-Ion battery pack and demonstrating the basic working principles of a battery management system. The project includes simulation, terminal output observation, and physical hardware implementation on a pertinax board.

The circuit was first tested in Proteus, and then transferred to a pertinax board to create a more permanent and organized prototype.

## Hardware Implementation

The hardware part of the project was transferred onto a pertinax board to make the circuit more stable and suitable for demonstration. This stage helped improve the physical structure of the project and provided hands-on experience in soldering, wiring, and circuit organization.

## Simulation and Testing

Proteus was used to simulate and observe the circuit behavior before and during the hardware implementation process. Terminal output screenshots were also used to verify the system response and show the monitored values.

This project was developed for educational purposes. It demonstrates the basic principles of a microprocessor-based battery management system and should not be used directly in high-power or commercial battery applications without further safety improvements and professional validation.
